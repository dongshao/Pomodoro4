# Pomodoro●doro

一个极简、优雅的番茄钟专注计时器，纯前端单文件实现。

## 功能

- **三种模式**：专注（25分钟）、短休息（5分钟）、长休息（15分钟）
- **SVG 圆形进度环**：带平滑动画的倒计时可视化
- **会话追踪**：4个番茄为一个完整周期，自动切换到合适休息模式
- **设置面板**：自定义各模式时长，启用自动开始
- **桌面通知**：计时结束时触发浏览器通知（需授权）
- **环境音效**：轻柔白噪音，帮助保持专注
- **暗色模式**：一键切换亮/暗主题
- **键盘快捷键**：`Space` 开始/暂停，`R` 重置

## 部署到 GitHub Pages

1. 将代码推送到你的 GitHub 仓库 `main` 分支
2. 进入仓库 **Settings → Pages**
3. Source 选择 **Deploy from a branch**，Branch 选 `main`，目录选 `/ (root)`
4. 点击 **Save**，等待约1分钟即可访问

## 本地预览

直接用浏览器打开 `index.html` 文件即可，无需任何构建步骤或服务器。
